# 🧠 Brain Tumor Distribution Analysis (Power BI Dashboard)

## 📌Overview

This project presents an **interactive Power BI dashboard** for analyzing brain tumor patient data, focusing on tumor types, demographic patterns, and anatomical distribution.

The dashboard enables users to explore key insights such as:

* Tumor type distribution (Glioma, Meningioma, Ependymoma)
* Age group trends
* Regional tumor occurrence in the brain
* Yearly diagnosis trends
* Tumor size and patient demographics

Brain tumors such as glioma and meningioma are among the most commonly studied tumor types in MRI datasets, widely used for medical analytics and classification tasks.

## 🎯Objectives
* Analyze patient-level tumor data using Power BI
*Identify high-risk age groups and regions
* Visualize trends in diagnosis over time
* Build an interactive healthcare analytics dashboard

## 📂 Dataset
* Source: Public brain tumor dataset (MRI-based / simulated tabular dataset)
* Format: CSV
* Key fields:
  * Patient ID
  * Age
  * Tumor Type
  * Tumor Size
  * Brain Region
  * Diagnosis Year
  * Tumor Grade

## 📊 Dashboard Features
1. KPI Cards
 * Total Patients
 * Average Age
 * Average Tumor Size
 * Most Affected Brain Region
2. Tumor Type Distribution
 * Donut chart showing proportions of:
  * Glioma
  * Meningioma
  * Ependymoma
3. Top Affected Brain Regions
 * Bar chart highlighting regions like:
 * Cerebellum
 * Brainstem
 * Temporal lobe
4. Age Group Analysis
  Identifies most affected group (e.g., 41–50)
5. Time Trend Analysis
  Line chart showing annual diagnosis trends
6. MRI Visualization
  Sample MRI scan included for contextual understanding
7. Interactive Slicer
  Filter by tumor grade (Confirmed / Unconfirmed)
